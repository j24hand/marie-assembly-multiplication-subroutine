/ ---------- MULTIPLICATION SUBROUTINE ----------
/ multiplies T1 * T2 using repeated addition
Mult, HEX 0
      Clear           / clear AC to 0
      Store Result    / initialize Result = 0
Loop, Load Result     / load current result
      Add T1          / add T1 each loop
      Store Result    / store updated result
      Load T2         / load loop counter
      Subt One        / subtract 1 from counter
      Store T2        / store updated counter
      Skipcond 400    / skip next instruction if AC = 0
      Jump Loop       / repeat loop until T2 becomes 0
      JumpI Mult      / return from subroutine

/ ---------- POSITIVE CHECK SUBROUTINE ----------
/ returns only when AC > 0
CheckPositive, HEX 0
CheckLoop, Skipcond 800     / if AC > 0, skip next instruction
           Jump BadInput    / if value is not positive, go to bad input
           JumpI CheckPositive / return if input is positive

BadInput,  Output 0         / output invalid value / error case
           Halt             / stop program if input is not positive
          / Store Temp
         /  Load Temp
         /  Jump CheckLoop
